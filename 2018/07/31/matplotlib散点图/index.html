Matplotlib 散点图

SepalLength SepalWidth PetalLength PetalWidth Name
5.1 3.5 1.4 0.5 Iris-setosa
6.1 2.1 2.1 0.1 Iris-versicolor
7.1 1.1 1.1 0.1 Iris-virginica

每种Name仅给出一组样例。
用数据来画一个花瓣（petal）和花萼（sepal）长度（lenght）的散点图。
首先利用 pandas.read_csv() 读取文件内容，返回一个 DataFrame。

创建一个字典，字典的index为Name，其中使用nuique去重。value为三种颜色，以用来区分不同的Name。

使用 groupby 将数据以Name进行分组，该函数返回一个 tuple, 有两个元素，一个为Name,另一个为Name下的数据元组（groupby 的作用可借鉴SQL语句中的group by）,两个元素分别赋给species和group。并通过循环进行画点。

要绘制单个点，使用matplotlib.pyplot.scatter函数，传入横坐标（group[‘PetalLength’]），纵坐标（group[‘SepalLength’]），并设置颜色，label, alpha 为透明度（0~1），marker 为点的形状，参数取值可参考文档，默认为‘o’圆圈。

legend 设置图例，然后指明横坐标与纵坐标的名称，然后show。
