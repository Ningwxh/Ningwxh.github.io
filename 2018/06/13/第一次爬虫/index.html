python 爬虫第一站

利用requests 和 BeautifulSoup 库进行批量爬取小说（比较文字是最好爬的手动滑稽）
网上随便找个小说网址

https://www.ppzuowen.com/shashibiya/

F12发现作品集的url 都在 ul 的 li 标签里的href属性里。
右键copy该标签的selector路径

rs_zp = requests.get(‘https://www.ppzuowen.com/shashibiya/')
rs_zp.encoding = ‘gbk’
soup_zp = BeautifulSoup(rs_zp.text, ‘lxml’)
zp = soup_zp.select(‘body > div.warp960.mt10.fix > div.warp700.fl > div > ul > li > a’)

requests 请求网页并设置编码格式，因为requests库默认‘ISO-8859-1’编码, 在使用 rs_zp.text 时会出现乱码，所以使用 rs_zp.encoding 改变其编码方式
创建 BeautifulSoup 对象并将其解析成 lxml, 利用 select 通过标签路径得到 li 标签组里的所有的a标签，返回的是一个元素为 tag 类型 的列表。
select 还可以通过 id ，类名（classs属性），甚至组合查找（用空格隔开）
现在我们要把 a 标签里的 href 属性和内容提取出来（内容是作品的名字，我们可以用来命名文件名进行分类）。

list_zp_url = []
list_zp_name = []
for i in zp:
list_zp_url.append(i.get(‘href’))
list_zp_name.append(i.string)

创建两个空列表然后用循环将目标加入相应列表中。此处用了tag对象的get方法可以获取属性的值，string方法可以获取标签内容，注意i.string返回的类型是并非 str, 用 type 方法可发现返回 NoneType 。
以同样的方法抓取每个作品的章节路径与章节名。

for i_zp in range(list_zp_url.len()):

#进行 str 拼接获取完整url
rs_ml_url = ‘https://www.ppzuowen.com' + list_zp_url[i_zp]
rs_ml = requests.get(rs_ml_url)
rs_ml.encoding = ‘gbk’
soup_rs_ml = BeautifulSoup(rs_ml.text, ‘lxml’)
ml = soup_rs_ml.select(‘body > div.warp960.mt10.fix > div.warp700.fl > div > ul > li > a’)
list_ml_href = [] #目录的href
list_ml_target = [] #章节名称
for i in ml:
list_ml_href.append(i.get(‘href’))
list_ml_target.append(i.string)
print(list_ml_href)
print(list_ml_target)

抓取内容并写入txt文件

for i in range(list_ml_href.len()):

url = ‘https://www.ppzuowen.com' + list_ml_href[i]
rs = requests.get(url)
time.sleep(2)
rs.encoding = ‘gbk’
soup = BeautifulSoup(rs.text, ‘lxml’)
data_text = soup.select(‘body > div.warp960.mt10.fix > div.articleBody.articleContent1 > div.articleContent > p’)
string_text = ‘’
for j in data_text[:]:
if j.string == None:
continue
else:
string_text += str(j.string)
print(string_text)
with open(‘本地文件夹路径’ + str(list_zp_name[i_zp]) + ‘.txt’, ‘a’, encoding=’gb18030’) as f:
f.write(list_ml_target[i])
f.write(‘\n’)
f.write(string_text)
f.write(‘\n’)
f.close()
print(list_ml_target[i])
print(list_zp_name[i_zp])

重复代码太多了，可写成类的进行优化。
完。
